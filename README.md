# Visual Tutor Live

Visual Tutor Live is a multimodal AI learning agent that helps users understand screenshots, notes, charts, and slides with simple explanations and key takeaways.

## Project status
Early hackathon MVP in progress.

## Problem
Many users struggle to quickly understand screenshots, study notes, charts, and slides. Traditional chat tools usually require the user to describe the visual content manually before getting useful help.

## Solution
Visual Tutor Live allows a user to upload an image or screenshot and receive:
- a simple explanation
- a short summary
- 3 key takeaways

## Planned MVP scope
- Upload an image or screenshot
- Analyze the visual input with Gemini
- Return a clear explanation
- Return 3 key takeaways

## Tech stack
- Gemini
- Google Cloud
- GitHub
