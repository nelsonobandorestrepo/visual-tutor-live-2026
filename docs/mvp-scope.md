# MVP Scope

## Project name
Visual Tutor Live

## One-sentence definition
Visual Tutor Live is a multimodal AI learning agent that analyzes uploaded screenshots, notes, charts, and slides and returns a simple explanation with key takeaways.

## Core user flow
1. User uploads an image or screenshot
2. The backend sends the visual input to Gemini
3. The system analyzes the content
4. The app returns:
   - what it sees
   - a simple explanation
   - 3 key takeaways

## Out of scope for this MVP
- Voice interaction
- Memory
- Authentication
- Database
- Advanced UI
- Multi-agent workflows
- RAG
- Avatar generation
